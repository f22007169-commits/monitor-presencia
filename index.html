🚀 Instrucciones para Ejecución Local

Para levantar la aplicación completa con todas sus funciones de backend en tu entorno local, ejecuta los siguientes comandos en tu terminal:

  1. Instalar Dependencias: pip install -r requirements.txt
  2. Ejecutar el Servidor: Ejecuta el archivo INCIAR.bat o directamente usando python app.py
  3. Acceder a la App: Abre tu navegador e ingresa a http://localhost:5000
